## 📱 How to Use

### Editing an Existing Blog Post

1. **Go to Blog Manager**
   - Visit: https://mindcentre.abacusai.app/admin
   - Click "Open Blog Manager" (purple button)

2. **Find Your Post**
   - Use the search bar to find posts by title, date, or author
   - Browse through the list with featured images

3. **Click "Edit"**
   - Opens the visual editor
   - Edit any field (title, date, author, content, image)

4. **Edit Content**
   - Use the rich text toolbar at the top
   - Format text like in Microsoft Word
   - Add images, videos, links

5. **Preview**
   - Click "Preview" to see how it looks
   - Opens in a new tab

6. **Save**
   - Click "Save Changes" button
   - See success message
   - Changes are live immediately!

### Creating a New Blog Post

1. **Click "New Post"**
   - From the blog manager page

2. **Fill in Details**
   - **Title**: Enter your blog post title
   - **Slug**: Auto-generated (can edit)
   - **Date**: Pick publication date
   - **Author**: Your name
   - **Featured Image**: Upload/paste image URL
   - **Content**: Write your article using the rich editor

3. **Create Post**
   - Click "Create Post" button
   - Redirects to edit page
   - Post is now live!

### Deleting a Blog Post

1. **Find the Post**
   - In the blog manager list

2. **Click "Delete"**
   - First click shows "Confirm Delete" button
   - Click again to permanently delete
   - Can cancel if you change your mind

## 💡 Tips & Best Practices

### Writing Blog Posts
1. **Titles**: Keep them clear and descriptive (50-60 characters)
2. **Slugs**: Use lowercase, hyphens for spaces, no special characters
3. **Images**: Use high-quality images (recommended: 1200x630px)
4. **Content**: Break into sections with headings (H2, H3)
5. **Preview**: Always preview before saving

### Content Formatting
- Use **Bold** for emphasis
- Use *Italic* for quotes or subtle emphasis
- Use **Headings** (H2, H3) to organize content
- Use **Lists** for steps or bullet points
- Add **Links** to relevant resources
- Insert **Images** to break up text

### SEO Tips
- Include keywords in title and first paragraph
- Use descriptive headings (H2, H3)
- Add alt text to images (in the editor)
- Keep URLs short and meaningful
- Write meta descriptions (first 150 characters)

## 🔧 Technical Details

### What Was Built

1. **New Pages Created:**
   - `/admin/blog` - Blog list view
   - `/admin/blog/edit/[id]` - Edit existing post
   - `/admin/blog/new` - Create new post

2. **API Routes Created:**
   - `/api/blog/update` - Update existing post
   - `/api/blog/create` - Create new post
   - `/api/blog/delete` - Delete post

3. **Technologies Used:**
   - **React Quill** - WYSIWYG rich text editor
   - **Next.js 14** - Framework
   - **TypeScript** - Type safety
   - **Tailwind CSS** - Styling
   - **Lucide Icons** - Icons

### File Structure
```
app/
├── admin/
│   ├── page.tsx (main admin panel)
│   └── blog/
│       ├── page.tsx (blog list)
│       ├── edit/[id]/page.tsx (edit post)
│       └── new/page.tsx (new post)
└── api/
    └── blog/
        ├── update/route.ts
        ├── create/route.ts
        └── delete/route.ts

public/
└── blog-data.json (all blog posts)
```

## 🆘 Troubleshooting

### Can't see the blog manager?
- Clear browser cache (Ctrl+Shift+Delete)
- Hard refresh (Ctrl+F5)
- Visit: https://mindcentre.abacusai.app/admin

### Changes not saving?
- Check internet connection
- Look for error messages
- Refresh and try again

### Rich editor not loading?
- Wait a few seconds for it to load
- Refresh the page
- Clear browser cache

### Image not showing?
- Check image URL is correct
- Make sure image is uploaded/accessible
- Try a different image URL
